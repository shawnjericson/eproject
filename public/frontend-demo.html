How to Integrate Maintenance Detection

Include this script in your frontend webapp:

<script src="/js/maintenance-detector.js"></script>

Or initialize manually:

const detector = new MaintenanceDetector({ apiUrl: '/api/health', maintenanceUrl: '/maintenance', checkInterval: 30000, // Check every 30 seconds retryCount: 3 });

What Happens During Maintenance

  • ✅ Frontend detects maintenance mode via API calls
  • ✅ Shows maintenance notification to user
  • ✅ Automatically redirects to maintenance page
  • ✅ Continues checking for when maintenance ends
  • ✅ Admin CMS remains fully accessible

API Endpoints

  • GET /api/health - Health check (returns 503 during maintenance)
  • GET /api/status - Status check (always returns 200 with maintenance info)
  • GET /maintenance - Maintenance page
