React Maintenance Mode Integration

✅ System Status: Healthy

Method 1: React Hook (Recommended)

Use the useMaintenanceMode hook for fine-grained control:

// Install the hook import { useMaintenanceMode } from './js/react-maintenance-hook'; const App = () => { const { isMaintenanceMode, isChecking, error, manualCheck } = useMaintenanceMode({ apiUrl: '/api/health', maintenanceUrl: '/maintenance', checkInterval: 30000, retryCount: 3, autoRedirect: true }); if (isMaintenanceMode) { return <div>Maintenance Mode Active</div>; } return <div>Your App Content</div>; };

Method 2: React Component (Simpler)

Wrap your app with the MaintenanceDetector component:

import MaintenanceDetector from './js/react-maintenance-component'; const App = () => { return ( <MaintenanceDetector apiUrl="/api/health" maintenanceUrl="/maintenance" checkInterval={30000} autoRedirect={true} > <div>Your App Content</div> </MaintenanceDetector> ); };

Method 3: Axios Interceptor (For API-heavy apps)

Automatically detect maintenance mode in API responses:

import axios from 'axios'; const api = axios.create({ baseURL: '/api', timeout: 10000 }); // Add response interceptor api.interceptors.response.use( (response) => response, (error) => { if (error.response?.status === 503) { const data = error.response.data; if (data?.error === 'maintenance_mode') { // Redirect to maintenance page window.location.href = data.maintenance_page_url || '/maintenance'; } } return Promise.reject(error); } );

Installation Steps

Step 1: Copy Files

Copy these files to your React project:

  • public/js/react-maintenance-hook.js
  • public/js/react-maintenance-component.js

Step 2: Choose Integration Method

Select the method that best fits your app:

  • Hook: Best for apps needing fine-grained control
  • Component: Best for simple automatic handling
  • Axios: Best for API-heavy apps

Step 3: Configure Options

Customize the behavior:

  • apiUrl: Your API health check endpoint
  • maintenanceUrl: Your maintenance page URL
  • checkInterval: How often to check (default: 30s)
  • autoRedirect: Whether to auto-redirect (default: true)

Step 4: Test

Enable maintenance mode in admin panel and verify your React app redirects properly.

API Endpoints

Your React app will check these endpoints:

  • GET /api/health - Returns 503 during maintenance, 200 when healthy
  • GET /api/status - Always returns 200 with maintenance status
  • GET /maintenance - Maintenance page

Response Format

During maintenance mode, API returns:

{ "error": "maintenance_mode", "message": "Website is currently under maintenance. Please try again later.", "maintenance_page_url": "http://yoursite.com/maintenance" }
